    Step 5: Concept Taxonomy

    Overview

    This taxonomy organizes the 200 concepts into 12 balanced categories to facilitate navigation and understanding of the learning graph structure.

    Taxonomy Categories

    ID Category Name Description Target %
    FUND Fundamentals Core electrical concepts and laws 15-20%
    PASS Passive Components Resistors, capacitors, diodes 12-15%
    ACT Active Components Transistors and integrated circuits 10-12%
    IO Input/Output Sensors, buttons, LEDs, motors 12-15%
    BREAD Breadboarding Prototyping and circuit construction 10-12%
    MEAS Measurement Testing, debugging, multimeter use 7-9%
    PWR Power Systems Power supplies, regulation, efficiency 7-9%
    DIG Digital Logic Boolean logic, gates, truth tables 6-8%
    ADV Advanced Circuits Complex projects and applications 4-6%
    SKILL Practical Skills Wire management, component handling 5-7%
    CIRC Circuit Theory Series, parallel, analysis techniques 8-10%
    SYS System Design Integration and complete projects 3-5%
